# Definition of done

The AI-native workflow is done for a feature when all of the following are true:

- the feature spec reflects the delivered scope,
- the ADR is updated or explicitly not needed,
- the ExecPlan records the implementation history and current outcome,
- code and tests are in the repository when behavior changed,
- the relevant validation suites passed or the remaining gap is stated clearly,
- transient artifacts are cleaned up,
- PR handoff material exists for review.
