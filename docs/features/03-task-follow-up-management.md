# Task and Follow-Up Management

## Description

As a backoffice campus worker, I need to manage student-related tasks and reminders so that follow-ups are visible, assigned, and completed on time.

## Acceptance criteria

- Users can create tasks linked to a student record.
- Tasks can be assigned to the creator or another worker.
- Tasks support due date, priority, category, and reminder fields.
- Task statuses include open, in progress, completed, cancelled, and overdue.
- Users can view personal and team task queues.
- Overdue follow-ups are highlighted in the interface.
- The system prompts users when a student has no next action for a configurable period.
