# Interaction Logging

## Description

As a backoffice campus worker, I need to log every student interaction so that communication history, outcomes, and follow-up needs are visible across teams.

## Acceptance criteria

- Users can log interactions against a student record.
- Supported channels include email, phone call, in-person meeting, internal note, and SMS or message.
- Each interaction stores date and time, user, channel, summary, and outcome.
- An interaction can optionally include a follow-up action.
- Interaction history is visible in the student timeline or a dedicated history view.
