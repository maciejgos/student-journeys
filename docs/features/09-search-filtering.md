# Search and Filtering

## Description

As a backoffice user, I need fast search and filtering tools so that I can quickly find student records, work queues, and common operational views.

## Acceptance criteria

- Global search supports student ID, application ID, name, email, and phone number queries.
- Users can sort and filter records and queue views.
- Users can save filters for common operational views.
- Search and filters are available in the primary list and queue experiences where they are needed.
