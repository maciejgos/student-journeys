# Notifications

## Description

As a backoffice user, I need timely notifications about work changes so that I can respond to assignments, deadlines, escalations, and status updates without manual checking.

## Acceptance criteria

- The system notifies users about new assignments.
- The system notifies users about upcoming due dates and overdue tasks.
- The system notifies users about escalated cases.
- The system notifies users about status changes on records they own or follow.
- Notifications are available in-app, with optional email support handled as a later extension.
