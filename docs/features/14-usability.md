# Usability

## Description

As a non-technical backoffice user, I need the interface to be simple and efficient so that common administrative tasks can be completed with minimal training.

## Acceptance criteria

- The interface is usable by non-technical backoffice staff with minimal training.
- Common tasks can be completed within a small number of steps.
- Student records present the most relevant operational information first.
- Navigation and terminology align with campus administration workflows.
